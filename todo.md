## قائمة المهام لمشروع الموقع التعليمي للأمن السيبراني

### المرحلة 1: البحث وجمع المعلومات والموارد
- [x] البحث عن معلومات عامة حول الأمن السيبراني وأهميته.
- [x] البحث عن معلومات حول تخصصات الأمن السيبراني المختلفة (Red Team, Blue Team, GRC, Ethical Hacking, Incident Response, Malware Analysis).
- [x] البحث عن دورات وموارد تعليمية باللغتين العربية والإنجليزية لتخصصات الأمن السيبراني المختلفة من المنصات المذكورة (Cybrary, TryHackMe, Hack The Box, Coursera, Udemy المجاني، أكاديمية طويق، أكاديمية سطر، أكاديمية حسوب).
- [x] البحث عن أدوات وتطبيقات الأمن السيبراني (Wireshark, Burp Suite, Kali Linux, Splunk) وشروحاتها.
- [x] البحث عن تمارين ومختبرات افتراضية مجانية للأمن السيبراني.
- [x] البحث عن كتب إلكترونية، منصات تدريب مجانية، وقنوات يوتيوب عربية موثوقة في مجال الأمن السيبراني.
- [x] البحث عن الشهادات الدولية المطلوبة في الأمن السيبراني (CompTIA Security+, CEH, OSCP, CISSP) وما يناسب كل مسار.

### المرحلة 2: تصميم هيكل الموقع وتخطيط المحتوى
- [x] تحديد الأقسام الرئيسية والفرعية للموقع.
- [x] رسم خريطة الموقع (Sitemap).
- [x] تحديد بنية كل صفحة ومكوناتها.

### المرحلة 3: إنشاء المحتوى النصي والتعليمي
- [x] كتابة مقدمة عن الأمن السيبراني وأهميته.
- [x] إعداد قسم التقييم الأولي الذاتي للمستخدم.
- [x] كتابة أقسام التخصصات المختلفة (Red Team, Blue Team, GRC, Ethical Hacking, Incident Response, Malware Analysis).
- [x] إنشاء مكتبة المصادر والموارد التعليمية.
- [x] إعداد قسم الأسئلة المتكررة.
- [x] إنشاء قاموس المصطلحات.
- [ ] كتابة وصف لكل تخصص من تخصصات الأمن السيبراني.
- [ ] تجميع خرائط الطريق لكل تخصص مع روابط الدورات والأدوات والمشاريع.
- [ ] إعداد مكتبة المصادر (كتب، منصات، قنوات يوتيوب).
- [ ] إعداد قسم الأسئلة المتكررة وقاموس المصطلحات.
- [ ] إعداد دليل الشهادات الدولية.

### المرحلة 4: البحث عن الصور والموارد البصرية
- [ ] البحث عن صور ورسوم توضيحية مناسبة لكل قسم.
- [ ] التأكد من توافق الصور مع الهوية البصرية للموقع.

### المرحلة 5: تطوير الموقع والواجهة التفاعلية
- [ ] إعداد بيئة التطوير.
- [ ] بناء الواجهة الأمامية للموقع (HTML, CSS, JavaScript).
- [ ] التأكد من استجابة التصميم للموبايل والتابلت.
- [ ] دمج المحتوى النصي والصور في الموقع.
- [ ] برمجة التقييم الأولي الذاتي.

### المرحلة 6: اختبار الموقع وضمان الجودة
- [ ] اختبار جميع الروابط للتأكد من عملها وصحتها.
- [ ] اختبار وظائف الموقع التفاعلية.
- [ ] مراجعة المحتوى لغوياً وتقنياً.
- [ ] اختبار التوافقية مع المتصفحات والأجهزة المختلفة.

### المرحلة 7: نشر الموقع وتسليم النتائج للمستخدم
- [ ] نشر الموقع على استضافة مناسبة.
- [ ] تقديم رابط الموقع للمستخدم.
- [ ] تقديم تقرير موجز عن المشروع.
